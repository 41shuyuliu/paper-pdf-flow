# 插件本地模式快速开始

当前模式无需启动后端服务，直接在插件内完成 PDF 解析与 Markdown 生成。

## 1. 在 Chrome 中加载插件
1. 打开 `chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择目录：`paper_pdf_flow_skill/extension`

## 2. 使用流程
1. 打开插件弹窗
2. 选择一个 PDF 文件
3. 选择输出语言与生成模式
4. 可选填写输出文件名（必须以 `.md` 结尾）
5. 点击“开始本地生成”
6. 等待自动下载 Markdown 结果

## 说明
- 所有处理在插件弹窗本地执行。
- 当前解析引擎为打包到插件内的 `pdf.js` + 启发式文本规则。
