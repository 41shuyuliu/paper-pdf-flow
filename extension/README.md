# 插件本地模式快速开始

当前模式无需启动后端服务，直接在插件内完成 PDF 解析与 Markdown 生成。

## 1. 在 Chrome 中加载插件
1. 打开 `chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择目录：`paper_pdf_flow_skill/extension`

## 2. 使用流程
1. 打开任意网页，点击浏览器扩展按钮
2. 选择一个 PDF 文件
3. 可选填写输出文件名（必须以 `.md` 结尾）
4. 点击“开始本地生成”
5. 等待自动下载 Markdown 结果（固定输出为中文最终版）

## 3. 面板行为（最小开关规则）
- 扩展按钮点击行为：
  - 面板关闭时：打开并重置
  - 面板打开时：关闭
- 失焦、切换标签、切桌面、点击页面其他位置：保持当前状态，不自动开关。
- 页面刷新后默认关闭（不自动恢复）。

## 说明
- 所有处理在网页内常驻面板本地执行。
- 当前解析引擎为打包到插件内的 `pdf.js` + 启发式文本规则。
